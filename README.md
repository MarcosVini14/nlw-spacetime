## Sobre o projeto
O projeto é uma cápsula do tempo para exibir memórias em uma linha do tempo. Esse projeto é totalmente responsivo

## Tecnologias
Esse projto foi desenvolvido durante a NLW da Rocket Seat com as seguintes tecnologias

- HTML
- CSS
- Git e Github
